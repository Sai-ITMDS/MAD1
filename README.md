# Hospital Management System v2 (Full)

This is the enhanced HMS v2 with:
- Sample data (3 doctors, 5 patients, 10 appointments)
- Doctor availability (weekly slots stored as JSON)
- Sidebar layout for Admin & Doctor dashboards
- Chart.js for admin analytics
- Responsive Bootstrap + custom CSS

## Run

1. Create venv and activate
2. pip install -r requirements.txt
3. python app.py
4. Open http://127.0.0.1:5000

- username/email: admin
